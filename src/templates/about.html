About

2nd Hand is a responsive web-based solution for second hand goods shopping. It has been realised by two students of the Free University of Bolzano, Giulio Roman and Aurelia Pagano, as final project of the Advanced Internet Technologies course.

The application has been written in PHP, HTML 5, CSS using modern frameworks as AngularJS and Bootstrap.

It is possible to view offered products, add them to the cart and place an order.
